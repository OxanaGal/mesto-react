# Проект «Mesto»

Учетный проект «Mesto» — это интерактивная страница, на которой пользователи могут делиться фотографиями, удалять их и ставить лайки.

## Установка и запуск приложения

Установка зависимостей:

### `npm test`

Запуск приложения

### `npm start`
